search(passage_name)¶

Retrieves a specific passage directly from the Bible Gateway site. Note that the output is subject to the Bible Gateway implicit passage limit when sending the web request.

Parameters

passage_name (str) – Name of the Bible passage which is valid when used on www.biblegateway.com

Returns

Bible passage with preserved line breaks

Return type

str (list if self.output_as_list is True)

search_multiple(passage_names)¶

Retrieves a set of passages directly from the Bible Gateway site. Passages can be from different books. Note that the output is subject to the Bible Gateway implicit passage limit when sending the web request.

Parameters

passage_names (list) – List of Bible passages that are valid when used on www.biblegateway.com

Returns

Bible passages with newline separators for each set of passages

str (list if self.output_as_list is True)
